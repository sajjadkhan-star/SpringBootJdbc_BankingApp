# Spring Boot JDBC Banking Application

## 📌 Project Description
This is a Banking System application built using Spring Boot and JDBC.
It allows users to perform basic banking operations.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- View Account Details
- Update & Delete Account

## 🛠 Tech Stack
- Java
- Spring Boot
- JDBC
- MySQL

## ▶️ How to Run
1. Clone the repository
2. Open in IDE (Eclipse/IntelliJ)
3. Configure MySQL database
4. Run the application
